Sobre

Sou graduado em ciência da computação, com formação técnica em informática para internet e experiência na área de suporte técnico, manutenção de computadores e recuperação de dados de mídias de armazenamento.

Como trabalho de conclusão de curso da minha graduação, desenvolvi um estudo de caso, cujo objetivo foi realizar o treinamento e análise comparativa de várias redes neurais artificiais, com configurações diferentes, visando estimar quais parâmetros possuem maior impacto na precisão da classificação. Para isso, foi desenvolvida uma aplicação em Python que realizava o treinamento utilizando a biblioteca scikit-learn e utilizando paralelismo utilizando threads para aumento de performance.

Já como trabalho de conclusão de curso do técnico, desenvolvi um sistema de controle de pedidos, utilizando tecnologias web como HTML5, CSS3, JavaScript e PHP 7. O sistema foi projetado para ser usado em restaurantes e cantinas, onde o usuário poderia realizar o pedido e efetuar o pagamento, através do sistema, e efetuar a retirada do pedido no balcão, evitando filas.
